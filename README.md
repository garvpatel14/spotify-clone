# Music Player UI Clone

A simple, clean music player interface inspired by modern streaming services.

## Features

- 🎵 Sidebar navigation
- 🎧 Playlist grid layout
- ⏯️ Interactive player controls
- 📊 Progress bar with seek functionality
- 🔊 Volume control
- ❤️ Like button
- 📱 Responsive design

## How to Run

1. Open `index.html` in your web browser
2. Click on any playlist card to "play" a song
3. Use the player controls at the bottom to interact

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Gradients)
- Vanilla JavaScript

## Interactive Features

- Click playlist cards to select songs
- Play/pause button toggles playback
- Progress bar animates during playback
- Click progress bar to seek
- Volume control with mute toggle
- Heart icon to like songs

## Files

- `index.html` - Main structure
- `style.css` - Styling and layout
- `script.js` - Interactive functionality
